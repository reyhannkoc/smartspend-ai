# SmartSpend AI

SmartSpend AI is a modern responsive personal finance web application that helps users manage income and expenses and provides AI-supported financial recommendations.

## Technologies Used

- HTML5
- CSS3
- JavaScript ES6+
- React
- Vite
- LocalStorage
- Fetch API
- Chart.js
- Git & GitHub

## Features

- Add income and expense transactions
- Delete transactions
- Filter transactions
- Calculate total balance
- LocalStorage support
- API integration
- AI-supported financial recommendations
- Chart visualization
- Responsive design

## Installation

1. Install project dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

## Deployment

This project can be deployed easily using modern frontend hosting platforms such as Vercel or Netlify.

- Vercel: Connect the repository, import the project, and deploy with the default Vite build settings.
- Netlify: Connect the repository, set the build command to `npm run build`, and publish the generated `dist` folder.
